Runtime

面向过程

  1. 这个运行时系统就像一个操作系统一样：它让所有的工作可以正常的运行
  2. 这个库使得C语言有了面向对象的能力

  3. 实现

  4. 对象之间靠消息传递，其中消息之间要能通讯要遵守一个叫协议的东西
  5. isa是消息机制的协议
  6. 占多大，分配多大，返回地址
  7. Selector相当于门牌号，而Implement才是真正的住户（函数实现）

因为一个 ObjC 类本身同时也是一个对象,类对象的类型叫做元类，类方法就定义在这个地方

Associated Objects

  1. 通过分类可以添加属性，但是不能添加实例

AssociationsManager里面是由一个静态AssociationsHashMap来存储所有的关联对象的。这相当于把所有对象的关联对象都存在一个全局map里面。而map的的key是这个对象的指针地址（任意两个不同对象的指针地址一定是不同的），而这个map的value又是另外一个AssociationsHashMap，里面保存了关联对象的kv对。

IMP lookUp(id obj, SEL selector) {
Class c = object_getClass(obj);
while(c) {
for(int i = 0; i < c->numMethods; i++) {
Method m = c->methods[i];
if(m.selector == selector) {
return m.imp;
}
}
c = c->superclass;
}
return _objc_msgForward;
}
